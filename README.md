# Rock-Paper-Scissors
This is a basic Rock, Paper, Scissors game that I made to practice combining JavaScript, HTML, and CSS.


### ✨ Introduction

This project is a browser-based Rock, Paper, Scissors game featuring a simple and interactive graphical user interface (GUI). Players can select their move using buttons, and the computer responds instantly with a randomized choice.

The game is built using HTML, CSS, and JavaScript, making it lightweight and accessible on any modern web browser—no installation required.

### ✨ Features
  • Play Rock, Paper, Scissors against the computer

  • Interactive GUI with clickable buttons

  • Instant and clear game results (win, lose, tie)

  • Randomized computer selection

  • Fun win animation

### ▶️ Usage

Open the game in your web browser.

Click one of the buttons:

🪨 Rock

📄 Paper

✂️ Scissors

The computer will make its choice automatically.

The result will be displayed on the screen.

### 🛠️ Troubleshooting

#### Game not loading properly?

  • Make sure you are using a modern browser (Chrome, Edge, Firefox, Safari)
  
#### Buttons not responding?

  • Check that JavaScript is enabled in your browser

#### Confetti not loading?

  • Ensure you have an internet connection
  
  • make sure reduced motion is disabled
